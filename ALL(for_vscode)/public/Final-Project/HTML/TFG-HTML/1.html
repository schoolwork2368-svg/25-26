    • Getting Started

    Welcome to TerraFirmaGreg! This guide will help you get started with the modpack, covering installation, basic gameplay mechanics, and essential tips for new players.

    Installation

    To install TerraFirmaGreg, follow these steps:

    1. Ensure you have Minecraft Java Edition installed.
    2. Download and install the required version of Minecraft Forge.
    3. Download the TerraFirmaGreg modpack from the official website or a trusted source.
    4. Place the downloaded modpack file into your Minecraft 'mods' folder.
    5. Launch Minecraft using the Forge profile.

    Basic Gameplay Mechanics

    TerraFirmaGreg introduces several new gameplay mechanics that enhance the survival experience:

    • Realistic World Generation: The world is generated with realistic biomes, terrain features, and resource distribution.
    • Survival Elements: Players must manage hunger, thirst, and temperature to survive in the harsh environment.
    • Crafting and Building: New crafting recipes and building materials are available to create tools, structures, and machinery.

    Essential Tips for New Players

    • Start by gathering basic resources like sticks, stones, and food.
    • Explore your surroundings to find valuable resources and safe locations for shelter.
    • Experiment with crafting recipes to discover new items and tools.
    • Stay vigilant against hostile mobs and environmental hazards.

    With these basics covered, you're ready to embark on your adventure in TerraFirmaGreg! Explore, build, and survive in this challenging modpack.

    Survival Mechanics

    Players must manage several survival mechanics, including:

    • Health: Avoid damage from mobs and environmental hazards.
    • Hunger: Keep an eye on your hunger bar and eat food to maintain energy levels.
    • Thirst: Find and drink water to stay hydrated.
    • Temperature: Dress appropriately for the environment to avoid hypothermia or heatstroke.

    Controls

    All the games controls can be customized via Settings>Controls>Keybinds

    Moevement Controls

    • W/A/S/D: Move forward/left/backward/right.
    • Spacebar: Jump.
    • Shift: Sneak.
    • Ctrl: Sprint.

    Interaction Controls

    • Left Click: Interact with objects.
    • Right Click: Use tools.

    Inventory and Menu Controls

    • E: Open inventory.
    • Esc: Open game menu.

    Additional Controls

    • Q: Drop held item.
    • Shift+Right Click: Transfer items between hotbar, inventory, and storage containers.

    Configuration

    There are a wide variety of configuration options available in the game:

    • Graphics Settings: Adjust rendering quality, resolution, and visual effects.
    • Audio Settings: Control volume levels for music, sound effects, and voice chat.
    • Gameplay Settings: Modify difficulty levels, world generation parameters, and survival mechanics.

    Accessing Configuration Options

    To access the configuration options, follow these steps:

    1. Open the game menu by pressing the 'Esc' key.
    2. Select 'Options' from the menu.
    3. Navigate through the different tabs to find specific settings.

    Configuration File

    Configuration options can also be modified directly in the configuration file.

    NOTE: The configuration file is located at Documents>TerraFirmaGreg>config.json

    Make sure to back up the configuration file before making any changes.

    The configuration file names

    There are multiple configuration files available in the game:

    • TFCOptions.cfg: This file contains general game settings.
    • TFCCrafting.cfg: This file contains crafting settings and controls recipes.
    • TFCOre.cfg: This file contains ore generation settings.

    TFCOre.cfg

    This file is used to determine the world generation of ores. Other mod ores can be added to this file to have them generate with TFC's terrain. Note: This config file can only be used to add ores from mods that use standard blocks, and not tile entities for their ores. Mods that use tile entities for their ore blocks, such as GregTech, cannot have their ores generated using TFC. Standard block metadata does not go higher than 15, so if the oreMeta is a larger number than that, the ore is a tile entity, and will not work.

    The stone type the ore can spawn in. Can be a category or a specific rock. Entries should be all lowercase, with each new entry on a new line.

    The metadata (damage value) of the ore. Must be between 0 and 15.
    oreName terrafirmacraft:Ore1
    The block name of the ore.
    rarity 100
    Veins will spawn in 1 out of X chunks.

    The size of the vein. Choices are large, medium or small.
